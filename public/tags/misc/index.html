LaTeX2e

I have been entertaining myself (not others) by writing all my technical docs in LaTeX2e lately. The problem is that there is a dearth of quality documentation on the web. I get the impression that because LaTeX2e is geared towards producing beautiful print output the people who work with it are loath to produce documentation for it that is at once compendious, and in HTML. Come on guys! The typesetting language is IMHO unparalleled, and I absolutely love to use it.
